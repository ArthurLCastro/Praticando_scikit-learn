# Iris

Definindo um modelo de IA utilizando o MLPClassifier do Scikit-learn para prever a classificação de uma flor de Iris dentre 3 tipos (setosa, versicolor e virgínica) a partir de 4 características (largura e comprimento das pétalas e sépalas)

- Base de dados utilizada: [The Iris Dataset](https://scikit-learn.org/stable/auto_examples/datasets/plot_iris_dataset.html)

> *Nota*: <br> O código [mlp_sklearn_iris.ipynb](./mlp_sklearn_iris.ipynb) trata-se de uma adaptação do código "Using MLP Classifier in scikit-learn for Iris Dataset Classification" do Prof. Eduardo Sodré disponível em https://www.arnepy.com/exercicios-de-machine-learning
